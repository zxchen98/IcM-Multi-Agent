# Reuse - node or pipeline not reused

## Overview

The node is not reused or the pipeline is not reused. The customer wants to know why it is no reused or why it is reused.

## Solution

Root cause: The customer not meet the criteria of reuse.

Solution: Please compare the state of pipeline or node to this document: https://learn.microsoft.com/en-us/azure/machine-learning/how-to-debug-pipeline-reuse-issues?view=azureml-api-2
